# AI 商业化与价值捕获

> AI 的钱从哪里来、被谁拿走：模型是不是商品、价值落在哪一层、capex 的上限、以及"AI 像电力还是像社交媒体"。这是把技术能力翻译成经济结果的一层。

## 核心命题：模型商品化，价值向上游转移

### Benedict Evans（美，独立分析师/前 a16z，2026-06）
来源：[a16z 访谈](../videos/20260608-a16z-benedict-evans-ai-economics.md)
- **"基础模型不是产品，chatbot 不是产品，价值会在更上层。"** 论证：模型间**无网络效应、无可持续差异化杠杆**，差别只在花钱意愿；chatbot 是受限的 v1 UI；模型厂造不出上层全部软件（00:17–00:22）。
- 模型厂更像 **hyperscaler / 半导体 / 电信**：低层基础设施，无 leverage、无控制权，而非 Windows/iOS 那样能决定"什么被造出来"并捕获价值（00:20–00:22）。
- **移动数据这面镜子**：电信集体年收入约 1 万亿、capex 约 2000 亿、股价 20 年不动，"所有酷东西都是别人造的"——宏大昂贵、使用爆炸、改变生活，却没赚到钱，因价值全部上移（00:11–00:14）。
- **当下高价是"极端失衡"而非常态**："需求无限"不代表不会回到价格均衡；移动数据 15 年涨 1500–2000 倍照样打价格战，因本质是商品、开发者来回切换（00:22–00:23）。他谨慎地不下断言："不是我知道它们会变商品，而是这条推理链指向商品化，请解释为什么不会。"（00:54–00:55）

### Steven Sinofsky & Sema Amble（美，a16z，2026-07）
来源：[Software in the Age of Agents](../videos/20260707-a16z-software-in-age-of-agents.md)
- 对 Evans"价值上移"的**微观补充**：价值确实在上层，但上层是**难以复制的组织逻辑**，不是模型。**"vibe code 一个 SAP"严重低估**——SAP 装几年是因为它被定制到贴合公司实际怎么运转，数据在哪个库不重要、逻辑才重要（00:17–00:21）。
- **软件护城河 = 例外处理 + 收钱 + 没人设计过的怪黏性**："最黏的软件就是正在被用的软件"；企业里几乎所有有趣的事都是例外，而例外只在人脑里、不在字段里（00:11–00:16、00:29–00:30）。
- **创业机会在"两个巨头之间的缝"**：巨头在剧变期唯一不做的是扰动现有产品线，只会"把 AI 螺栓拧到侧面硬扛"；企业软件唯一真网络效应在公司内部（chat 正像当年 Excel 那样病毒式扩散）（00:53–00:59）。

### Freda（美/中，Altimeter，2026-05）：Token 经济学与"负向滚雪球"修正
来源：[张小珺访谈](../videos/20260518-zhang-xiaojun-freda-investment-2.md)
- **Token 不是可比的计量单位**（[00:02:03]）：关键是 Token per Task——同一任务不同模型消耗可差几十到上百倍（精练代码 vs 啰嗦代码、隐藏 reasoning tokens、agent workflow 放大）。按 Token 收费必然转向按效果收费（Sierra 客服模式：解决问题才收费）。
- **"负向滚雪球"修正**（[00:19:23]）：Dario 原框架低估了需求弹性——Anthropic 2026 年初需求爆发证明收入增长斜率可以比训练成本增长更陡，公司会突然盈利。练模型的成本占收入比例＜推理毛利率，公司就能赚钱。
- **模型层不是 SaaS**：SaaS 是 subscription（按人头、封顶）→ 客户倾向只用一家；模型是 usage-based → 鼓励同时用多个 vendor、随时切换。OA 和 Anthropic 因此不是传统意义上的竞争。
- **Code in TAM 被严重低估**（[00:17:19]）：从 100 亿美元（400 万 developer × $200）→ 真实 TAM 是"任何可以被计算机操作的事情"。Cowork 两个人做出来，同等规模时增长斜率超 Claude Code。
- **每个 GW 模型收入被系统性低估**（[00:22:28]）：年底 Anthropic/OA 各 ~5 GW，明年 ~10 GW。100 亿/GW 保守，500 亿也非不可能。

### 雨森（中，真格基金，2026-05）：Return 三段链与 Agent 网络效应
来源：[张小珺访谈](../videos/20260527-zhang-xiaojun-yusen-vc-2.md)
- **Return 三段链**（[00:24:16]）：投入→产出→结果。Anthropic 收入不是最终回报，是客户的投入。"这个问题被推迟了，没被解决。"水槽比喻：上面吃螃蟹的人>下面发现不好吃走了的人→ARR涨，但需堵住漏水。
- **Harness=OS，模型=CPU**（[00:53:15]）：Harness 处理底层（agentic loop/memory/sandbox），模型可插拔。用户忠诚于 Harness 非模型——"在里面救活自己的小龙虾，因为记忆在里面，但模型可以换（Kimi：90分表现20分价格）"。
- **Agent 网络效应**（[01:00:20]）：个性化 context 沉淀后，同一任务不同 agent 产出不同→有理由互相雇佣→agent marketplace。"1000块 token 费 + 9000块知识溢价。"
- **AI 应用创新多来自第三方**（[00:49:12]）：Wide Research、Memory MD、IM 插入、心跳机制——都不是模型公司做的。"模型公司认为不本质的细节，对用户很本质。"

### Peter Steinberger（美/奥，OpenClaw 作者，2026-02）：app 消亡、开源可持续性危机

来源：[OpenClaw 访谈](../videos/20260212-lex-openclaw-steinberger.md)

- **"每个 app 现在都是一个很慢的 API，不管它愿不愿意"**：agent 可走浏览器（Playwright + 住宅 IP）绕过封锁——"你没有让不可能的事发生，只是让它变慢了"。预计**杀死 80% 的 app**（MyFitnessPal/Sonos/日历类首当其冲：agent 有更全的 context 能做更好的决策）；app 要么变 agent-facing API 要么消亡（02:43–02:57）。这是 Sinofsky "headless 软件"论的激进版：不是软件变 headless，而是多数 head 直接不需要存在。
- **新价值层**：给 agent 发"零花钱"预算、rent-a-human 类服务、agent marketplace——与[戴雨森](../people/dai-yusen.md)的 agent 网络效应/marketplace 判断跨洋呼应（02:54–02:56）。
- **开源可持续性危机的一手数据**：OpenClaw 每月倒贴 1–2 万美元（赞助全转依赖库作者）；连"人人都用"的 **Tailwind 都裁员 75%**——agent 时代没人访问官网，流量入口的租值蒸发（02:20–02:22）。为 Evans 的"价值上移"补充一个受害者层：**免费开源基础设施的隐性商业模式（官网流量/雇主品牌）被 agent 掐断**。
- 变现路线的现实选择：拒绝 VC 与创业（商业版会与开源版利益冲突），选择"加盟大厂 + 项目保持开源"（Chrome/Chromium 模式，Meta vs OpenAI 二选一）——个人 agent 这一层的价值捕获，最终仍向模型厂/平台厂收敛（02:18–02:34）。与广密"壳公司窗口会被模型公司关闭"同向。

## capex 的物理上限

### Benedict Evans（2026-06）
- 微软/Meta/Google 今年 capex 均超营收 **50%**（电信才 15–20%），四大指引约 **7000 亿美元/年**——与油气同量级，"很多钱但非天量"（00:48–00:49）。
- 但有硬顶："**不可能一年花 10 万亿美元，因为世界上没有 10 万亿可花**"；1.5 万亿只能借、不可持续（00:49）。贯穿全场的一条线：**模型只相关 3–9 个月，却要花数十亿造，还得永远追下一个**（00:50）。
- 详见 [AI 算力与基础设施](ai-infrastructure.md)。这与 Imas & Trammell 的"付给算力的份额是升是降"是同一枚硬币的供给侧。

## "AI 像电力，还是像社交媒体？"——价值捕获的分叉

### Alex Imas & Phil Trammell（美，AGI 经济学，2026-06）
来源：[Dwarkesh 访谈](../videos/20260604-dwarkesh-imas-trammell.md)
- 这是本主题的**经济学骨架**：**AI 是电力（普惠、租值扩散、人人被 index）还是社交媒体（租值归平台、集中）**，取决于"开源模型落后前沿 6–9 个月"是否成立（01:00–01:10）。
- 对不在生产链上的国家：建议优先"**买 AGI 的 index**"而非重训——但若回报集中在私有 OpenAI/Anthropic 就难以 index（00:25–00:29）。
- 与 Evans 呼应：Evans 说"模型是商品 → 价值上移到应用层"，Imas·Trammell 追问"那被捕获的价值最终归谁"——若模型真商品化、开源紧咬前沿，则更像电力；若前沿厂靠资本壁垒拉开差距，则更像社交媒体。

## 中方视角：AR 爆发、token usage > DAU、壳公司与价值定价

### 广密（硅谷投资视角，2026-04）

来源：[广密·全球大模型季报第 9 集](../videos/20260415-zhang-xiaojun-guangmi-llm-quarterly-9.md)

- **coding 带来 AR 爆发**：Anthropic 官宣 AR 超过 OpenAI；**头部一两百万 coding 用户收入 > OpenAI 五六千万订阅**——"c 端没有 coding/agent 大"（00:09:14–00:10:18）。年底两家 AR 或达 800–1000 亿美金、明年奔 2000 亿，"已成新 MAG7"。
- **价值指标从 DAU 转向 token usage**（尤其塔尖超级开发者）；一个可跑正的微观指标：**"消耗 100 美金 token，能不能赚到 110"**（00:20:23、01:20:14）。
- **壳公司吃技术溢出红利、属阶段性窗口**：Cursor/Manus 若模型公司不开放最强 API（优先做自己的 agent 产品）就危险；"模型公司作为大哥也想吃这块肉，就看谁速度快"——今天模型公司更快（00:37:35–00:39:36）。这是对 Benedict Evans"价值上移到应用层"的一个反向修正：**当模型公司选择不溢出，应用层的价值捕获窗口会被主动关闭**。

### 罗福莉（中方 research lead，2026-04）：定价从成本转向价值、模型即产品

来源：[罗福莉访谈](../videos/20260424-zhang-xiaojun-luo-fuli-agent-paradigm.md)

- **定价逻辑从成本转向价值**：Flash 按架构成本优势定价（最低价最高速）；**Pro 抛弃成本定价、改为按"模型最终产生的价值"定价**——后训练做得好、更懂框架，就是溢价空间（01:40:52–01:41:55）。
- **模型即产品**："原来不想做产品，但模型 + agent 架构本身就变成了一套新产品，产品力反而更强"；"现在模型公司好像没有壁垒/边界了"（03:14:45）。与广密"模型公司想自己接住溢出红利"从供给侧呼应——价值正从"卖 API token"转向"卖 agent 产品/价值"。

## 谁捕获价值：C 端 vs enterprise 的中美分野

### 姚顺宇（中，Google DeepMind，2026-05）
来源：[张小珺访谈](../videos/20260511-zhang-xiaojun-yao-shunyu.md)
- **模型层"knowhow 已无秘密"**：中国模型发得快"说明这道题对所有人都变简单了"（03:06–03:08）——从供给侧印证 Evans"模型无差异化"。
- **价值捕获的中美分野**：中国强 **C 端**（能想出"很间接却能滚利润"的复杂产品：抖音免费视频 + 广告 + 直播 + 电商）；美国强**直接的 enterprise 效率软件**（写码成本 150 卖 200，直接挣 50）（03:24–03:27）。这与 Evans/Sinofsky 聚焦的美国企业软件价值捕获是同一问题的两种商业文明答案。详见 [中美 AI 生态对照](china-us-ai.md)。

### Raschka & Lambert（美，2026-01）
来源：[Lex #490](../videos/20260131-lex-state-of-ai-2026.md)
- **serving 成本 >> 训练成本**：DeepSeek 预训练传闻约 500 万、OLMo3 约 200 万，但服务上亿用户是数十亿美元级——这是价值捕获的成本侧现实，也解释了模型为何反而变小（00:51–00:53）。
- **"一个模型统治一切"的梦在死** → 走向专用化：金融/法律/医药用私有数据的定制模型是大生意（03:14–03:16）——与 Sinofsky"价值在难复制的组织逻辑/私有数据"一致。

### Andrew Feldman（美，Cerebras，2026-05）：快 AI 催生新商业模式
来源：[Cerebras 访谈](../videos/20260521-no-priors-cerebras-feldman.md)
- **"速度不是让现有商业模式变好一点"**：Netflix 从寄 DVD 变电影公司——不是更高效地寄 DVD，而是开辟全新生意；AI 现在只替代看得见的（编码/设计/SaaS 工具），围绕它**重组工作方式**后才有新商业模式与生产力跃迁（00:28–00:29）。与 Sinofsky/Evans"价值上移"互补：Feldman 从**速度**这一维度指出上移后会长出"现在还看不见的"生意。
- **纯 AI play 上市**：Cerebras 是唯一 100% 营收来自 AI 的标的，"从企业青春期毕业到成年"；OpenAI/Anthropic/Databricks 能在私募拿到公开市场估值，改变了传统 4 年 IPO 时间线——是价值捕获在资本市场侧的新形态（00:20–00:23）。

### Lila Sciences（美，neo-lab，2026-07）：模型即价值、"零 FTE 虚拟创业"
来源：[Lila 访谈](../videos/20260716-latent-space-lila-sciences.md)
- **模型本身才是 moat、不冲临床（neo-lab）**：实验室是 token 生成器/数据护城河；回应"有数据训模型、有了数据又何必模型"——通用平台下**领域数据需求随广度下降、相邻领域可降到零**（00:27–00:29、00:40–00:41）。
- **"零 FTE 虚拟创业"新商业形态**：客户带明确问题、在平台上跑完整 program（平台费+试剂+分成），可并行成千上万个虚拟创业；一个 in vivo CAR-T ≈ 5 年 biotech/6 个月/10% 成本——与戴雨森"one person company"、广密"AR 爆发"同属"AI 把创业单位成本压到极低"（00:49–00:52）。
- **材料"苦甜 scaling"**：只有能规模化的东西才 matter，从第一个实验就 supply-chain conscious、有 techno-economic agent 蹲在角落——这是把"价值捕获"前置进研发环节的具体做法（00:42–00:43、00:52–00:54）。

### Gavriel Cohen（NanoClaw，2026-06）：个人 agent 部署生意、需持续维护
来源：[NanoClaw 访谈](../videos/20260629-latent-space-nanoclaw.md)
- 早期采用者多是 VC/CEO"我搭好 setup 想推给全团队但不想当修 agent 的 IT"→ 催生企业 agent 部署生意（NanoClaw Co.）；**agent 不像传统软件"部署完 5 年不动"，模型底座在变必须持续升级**——价值捕获从"卖软件"转向"卖持续维护/托管"（00:14–00:18）。呼应 Steinberger"个人 agent 层的价值最终向平台厂收敛"。

## 硬件护城河的三层递进：消费电子视角（阳萌，安克创新CEO，2026-06）

来源：[张小珺访谈](../videos/20260608-zhang-xiaojun-yangmeng-anker.md)

阳萌提出了与软件/模型截然不同的硬件护城河框架（[02:14:34]），与本主题的"模型商品化→价值上移"构成对照：

1. **独特领先技术** — 但不是永久护城河："技术一定会遇到人的感知上限（Retina效应）——当分辨率超过人眼分辨力，再进步用户无感。"任何技术优势放在足够长时间里一定会被抹平。"（[02:15:34]）
2. **品牌信任** — "消费者可能需要很多次建立起对你的信任，做错一次就可能永久消失。"安克在欧美做到1万美金户用储能产品纯线上销售、首月1000+客户，是品牌信任的例证。
3. **使命愿景价值观+人** — "诺基亚有品牌、有信任，为什么不行了？最后还是要看公司是不是持续有为极致创新的一群人。"（[02:18:34]）

这实际是对Evans/Sinofsky"软件护城河=例外处理+组织逻辑"的硬件镜像：在硬件领域，**制造意外处理的不是代码逻辑，而是品质管控+全球供应链+品牌信任的累积效应**。

### 价值分配：创造者拿7，股东拿3

- 安克内部坚持的分配原则（[02:46:57]）。基础逻辑：好的消费电子公司有30%+的总剩余价值，股东合理回报~10%，因此7:3。
- "AI时代人的价值被N倍提高，但创造者依然是价值创造的主体，分配必须尊重这一规律。"这与Imas·Trammell的"电力vs社交媒体"之问不同——阳萌不是讨论价值被谁捕获，而是**主动设计捕获后的分配机制**。

### "大new人"与"小new人"

- **大new人**：造AI智能体的人（把业务认知/数据/流程沉淀到智能体）——少数。
- **小new人**：使用AI智能体的人——多数，效能也被N倍提升。
- 组织转型三件套（[02:53:01]）：人拥抱AI → 能力沉淀成智能体+智能体拉通 → 组织变革（拆出Scrum、重新分配决策权）。

### 与"软件即模型"的对照

阳萌的框架回答了Sinofsky"vibe code SAP被严重低估"在硬件领域的对应：消费电子价值链有200个岗位，AI不能缩短物理环节（电子工程师不会因AI就会画结构图），但可以让Scrum内部效率更高、跨Scrum协作更顺畅。**硬件护城河=物理世界不可压缩的环节 × 在这些环节上积累的组织knowhow**。

- **共识（截至 2026-07 已收录者）**：模型层正在商品化 / 无 winner-take-all（Evans、姚顺宇、Raschka & Lambert 三方独立印证）；价值会离开模型层，但"去哪一层、被谁拿"仍是开放问题。
- **分歧/张力**：
  - Evans 认为价值上移到**应用/工作流层**；Sinofsky 更强调上层是**难复制的组织逻辑与私有数据**（创业者常低估），二者互补而非冲突。
  - **价值最终集中还是扩散**是最大未决问题：Imas·Trammell 的"电力 vs 社交媒体"把它系于"开源能否紧咬前沿 6–9 个月"——直接把中国开源模型（见 [中美对照](china-us-ai.md)）放到了决定 AI 普惠性的关键位置。
- **待观察**：广告/电商（Evans 认为最可能被真正改造的万亿级 TAM）、专用垂直模型（Raschka & Lambert / Sinofsky）能否成为价值捕获的新落点。

## 应用层公司 vs 前沿实验室：架构非规模、model-agnostic、窄模型（All-In 双访谈，2026-07）

来源：[语音与法律](../videos/20260714-all-in-11labs-legora-voice-law.md) · [Cerebras 与 BFL](../videos/20260710-all-in-cerebras-bfl-open-source.md)

- **反复出现的紧张**：用前沿模型做产品、而前沿实验室（Anthropic/OpenAI）又想吃掉你的生意。四家应用层公司给出同构的护城河叙事——
  - **ElevenLabs（语音，$600M ARR）**：护城河是"**架构而非规模**（得改变模型怎么运作 + 很具体的未标注数据）"+ 千人标注 + 垂直化 + 生态；对前沿实验室做 **model-agnostic** 平台，专注"交互/沟通这一层"；承认对手在蒸馏其数据（[语音与法律](../videos/20260714-all-in-11labs-legora-voice-law.md) 00:25–00:29）。
  - **Legora（法律）**：法律服务 **$1 万亿/年**但软件只占约 4%（400 亿）；billable hour 被颠覆（企业把法务收回 in-house）；数据护城河**"要全不要 80%"**（litigator 需全部案例，是 power law 反面）；**不做通用法律智能模型、只做窄任务微调模型**；compliance 是货币（00:33–00:51）。呼应 [Genesis 的 narrow model 观](../videos/20260630-latent-space-genesis-diffusion-drug-discovery.md) 与 Sinofsky"上层是难复制的组织逻辑与私有数据"。
  - **Cerebras**：纯 AI play、**$250 亿 backlog**、推理的 Moore's law；"快 AI 造新商业模式"延续其 Netflix 类比（详见 [AI 基础设施](ai-infrastructure.md)）（[Cerebras 与 BFL](../videos/20260710-all-in-cerebras-bfl-open-source.md) 00:03–00:13）。
  - **Black Forest Labs**：开源 Flux + 闭源双轨；生成式影视**重排成本结构**（一部"比特币电影"$3000 万 vs 传统 $1.5 亿、否则不会被 green-light）；开源 + IP 授权 + 粉丝二创商业化（00:50–01:01）。
- **共同母题：垂直化 + model-agnostic + 窄模型**是应用层对抗前沿实验室的三件套；"独立第三方"逻辑（[Onyx 的安全买家心理](../videos/20260528-no-priors-onyx-security.md)：不信卖你产品的人认证产品）是同一叙事在安全领域的变体。这为本库"模型层商品化、价值去哪一层"的未决问题添了一批一线数据点：**价值捕获在"交互层/垂直数据护城河/合规与信任"，而非模型本身**。

## 中美对照

美方讨论（Evans、Sinofsky、Raschka & Lambert）几乎全部围绕**enterprise / SaaS 的价值捕获**——因为美国的钱在企业软件。姚顺宇则指出中国的价值捕获逻辑在 **C 端**（字节被严重低估、美国没公司能在 C 端与之竞争）。因此"AI 商业化"在两国是**两套不同的价值捕获范式**：美国问"agent 如何重构企业软件栈的租值"，中国问"如何用 AI 把 C 端超级 app 的利润滚得更大"。而 Imas·Trammell 的"电力 vs 社交媒体"之问，其答案很大程度上取决于中国开源模型能否让前沿能力平民化——这让中美生态竞争直接决定了全球 AI 的价值分配形态。
