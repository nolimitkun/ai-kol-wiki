# 戴雨森（雨森）

真格基金管理合伙人，早期投资人。二级市场偶像为 Stanley Druckenmiller。

## 投资风格

- **投人而非投赛道**：跟踪优秀个体多年，等他们创业就投，不管当时做什么方向
- **Strong opinions weakly held**：有强的观点但不被绑架，当原因变化就调整。频繁被打脸是幸福的——说明行业在变化、自己在成长
- **四类创始人框架**：小天才、老司机、科学家、操盘手。四力：学习力、创造力、领导力、意志力

## 代表性投资

- **Kimi**：早期投资，国内最关注 agentic coding 的基座模型公司
- **Manus、GenSpark**：第一波 agent Harness
- **刘松明、丁玲**：清华年轻学术创业者（世界模型/具身智能），跟踪多年后首轮投资
- **5G（灵巧手）、Slot（人+agent协作平台）**

## 核心观点

- **Return 问题未解决**：投入→产出→结果三段链。Anthropic 收入只是客户投入，最终回报待验证
- **Coding 是水平能力**：不是垂直赛道，能溢出到所有知识工作
- **Harness=OS，模型=CPU**：壳的价值在上升，用户忠诚于 Harness 非模型
- **Agent 网络效应**：个性化 context 沉淀后，不同 agent 产出不同→有理由互相雇佣→agent marketplace
- **下一个字节不会像字节**：不能把信息流模式搬进 AI 时代
- **AI Native 三阶段**：更多 agent → 适应人类世界 → agent 原生世界
- **组织变革类比**：蒸汽机→电动机的历史教训，AI 也需要组织配合才能释放生产力
- **思想健身房**：思考需要刻意练习，不能全外包给 AI
- **Agency 是最后堡垒**：Taste 也不保险，目前剩下的是问出好问题和做决定的 Agency
