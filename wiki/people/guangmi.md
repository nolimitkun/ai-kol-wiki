# 广密

- **背景**: 硅谷一线 AI 投资人 / 研究者，[张小珺](zhang-xiaojun.md)"**全球大模型季报**"系列的固定对谈嘉宾（连续三年、每季度一次，用季报形式追踪大模型技术进展与趋势）。长期身处硅谷前沿，与各 AI lab、研究者、超级开发者密切交流，提供"硅谷一线体感"下的一手观察。本人主线是投资模型公司。
- **来源出处**: [全球大模型季报第 9 集（2026-04）](../videos/20260415-zhang-xiaojun-guangmi-llm-quarterly-9.md)

> 广密的判断以"季报"形式连续记录，可追踪其观点演变——他自陈去年就反复强调"Coding, Coding, Coding"，2026 年 Q1 已如海啸兑现。与本库其他中方素材（[姚顺宇](yao-shunyu.md)、[朱邦华](banghua-zhu.md)、[阳萌](yangmeng-steven.md)）互补：广密提供的是**投资/战略/商业**视角的硅谷全景，而非技术内部视角。

## 立场与核心观点

- **AGI 三幕论**：Chatbot（第一幕）→ Coding Agent（第二幕，能干活）→ Automated AI Researcher（第三幕）。**Coding 是 AGI 的加速器**，"领先的 Coding 模型就像领先的 GPU"，模型公司不重视 coding 就会掉出第一梯队。
- **"语言即世界，代码即方案"**：只有语言与代码的泛化性被充分证明；coding 最先跑出来是因为 feedback loop 最短。
- **Opus 4.5→4.6 = GPT-3→GPT-4 级跨越**："AI 的基点时刻已到"；硅谷前沿 researcher/程序员"基本不写代码了"（AI 写、人审），部分 AI research 突破已由 Codex/Claude Code 而非人类带来。
- **token usage > DAU**：Anthropic 头部一两百万 coding 用户收入 > OpenAI 五六千万订阅；竞争指标转向 token usage 与塔尖用户；年底两家 AR 或达 800–1000 亿美金。
- **两条 AGI 路线**：C 端流量 vs 高价值任务；OpenAI/Google 抢 C 端窗口时严重误判 coding。
- **御三家画像**：Anthropic = 战略/专注/文化的胜利（top-down、all-in coding、重数据、防泄密、塔尖定价，风险在算力）；OpenAI = 阶段性被低估、50% 概率仍是最终 winner（bottom-up 可能出下个范式，但 SAM 摊大饼、"没有灵魂"）；Google = 最稳的追随者（算力/现金流最足、体系化，但 Gemini 3 被高估、误判 coding）。
- **挑战者**：META TBD 取代 XAI 成硅谷四号种子（但产品创新力不及中国团队）；XAI 因 Elon 战略摇摆、迷信"大力出奇迹"盲目 scale 参数而短期掉队（"bottleneck 是数据不是模型大小"）。
- **壳公司**：Cursor/Manus 吃模型公司技术溢出红利，属阶段性窗口；模型公司若不开放最强 API 则壳危险。
- **模型 = 新一代操作系统**：最领先的三五家模型 = global GDP 的 OS，每家 10 万亿美金。
- **Harness Engineering**：把 agent 当一等公民、给它搭环境；Harness 让非 SOTA/开源模型也能做高价值任务；思维从 2C/2B 转向 to 人类/to agent；Manus 是 Harness 鼻祖。
- **社会**：今年是白领失业痛苦之年（或 30% 岗位没了），"最牛 AI researcher 都担心 1–2 年后失业"；人类知识/智力变廉价；建议去创造、修炼 taste，one person company 或成常态。
- **投资**：只投能持续做 SOTA model 的公司；理想 portfolio = 三五家模型各 20% + 机器人 10% + AI for science 10% + agent infra 10%；机器人未来 6–18 个月或质变（中国团队有优势）。
