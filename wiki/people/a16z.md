# a16z（Andreessen Horowitz）

美国头部风投机构 Andreessen Horowitz 的官方播客/视频频道（@a16z）。不同于本库其它以固定主播为主的频道，a16z 频道是**机构声音 + 轮换嘉宾**的集合——合伙人、被投公司 CEO、以及独立分析师。本库收录时以**内容的分析质量**为准，跳过偏营销/名人访谈的期次。

## 已收录的声音

### Benedict Evans（独立科技分析师，前 a16z 合伙人）
- 硅谷最具影响力的科技通讯作者之一，年度演示 *AI Eats the World* 是行业风向标。
- **立场与关注点**：一贯"历史类比 + 拒绝确定性预测"。核心命题：**基础模型会沦为商品，价值向上游转移**；反复以移动数据产业为镜（宏大投资、改变生活、却没赚到钱，因价值上移）。强调当下高价是"极端失衡"而非常态、capex 有物理上限、生产力提升会被竞争抹平成消费者剩余。
- **呼应与分歧**：与 [Imas & Trammell](imas-trammell.md) 的"AI 是电力还是社交媒体"（价值捕获/集中）直接互补；"任务 vs 工作"呼应 [Jensen Huang](jensen-huang.md) 放射科案例；"模型无差异化"印证 [姚顺宇](yao-shunyu.md) / [Raschka & Lambert](raschka-lambert.md) 的"纸面趋同、无 winner-take-all"。

### Steven Sinofsky（a16z board partner，前微软 Windows/Office 负责人）
- 四十年企业软件一线经验（主导 Office、Windows 7/8）。
- **立场与关注点**：**企业里所有有趣的事都是例外处理**，例外只在人脑里、不在字段里；**生产力永远创造新工作**（长尾变长而非变短）；"vibe code 一个 SAP"严重低估企业软件的定制深度；MCP/middleware 是工程师的一厢情愿——没有软件愿意被抽象成哑数据库。
- **呼应与分歧**：放射科案例、反"固定馅饼"心态呼应 [AI 与就业](../topics/ai-and-jobs.md)；对 [Benedict Evans](#benedict-evans独立科技分析师前-a16z-合伙人)"价值上移"命题做微观补充——价值在上层，但上层是难复制的组织逻辑，不是模型。

### David（a16z 成长期投资负责人）
> ⚠️ 转录稿中仅以 "David" 出现、未播报全名。按业务与内容判断很可能是 **David George**，但本库未能从转录稿证实。

- **立场与关注点**：诊断的核心是一组对照——**Anthropic 与 OpenAI 每月新增收入超过 Meta/Google/Microsoft，而技术向真实经济的扩散不到 5%**。由此推出上界问题："钱从哪来"（财富 500 强年利润合计约 2 万亿）。
- **"必须在 token 路径（token path）上"**是他们看被投公司的**头号标准**，因为买方成本压力已经出现、不会给"上一代软件"加预算。
- **决定价值归属的最大变量是"模型公司的市场结构"，而他明确称之为"根本不可知"**：前沿两三家 → token 价格更高；五家 → 更低；**token 价格更低对整体经济更好**，因为不会有那么快重构劳动力的压力。
- **不是泡沫，因为供给受限而非需求受限**："我相当有信心说我们现在不在泡沫里。我没那么有信心说三年后不在。"唯一可能翻转的变量是**大幅变小的模型**（来自算法突破）。规模化数据中心容量要等到 2028 年底/2029 年初。
- **诚实的不确定性**："结果会大得多，**但要预测谁捕获那些价值，正在变得难得多**。"证据是 Forbes AI 50 榜单**一年内 40% 掉榜**。他复盘 a16z 自己三次立场翻转，现在的判断是"**模型公司正一路往上挤进应用层**"。
- 风险哲学：**明确拒绝低损失率**——"如果我们损失率低，那我们就没在承担足够的风险"；"糟糕的那一格是：**领域成了，而我们挑错了人**"。
- **呼应与分歧**：与 [Benedict Evans 的价值上移论](#benedict-evans独立科技分析师前-a16z-合伙人)方向相反——Evans 认为模型商品化、价值向上游走，此处认为模型公司自己在往应用层走；与 [All-In 的 ROI 之辩](../videos/20260711-all-in-ipo-token-roi-china-open-source.md)是同一辩论的投资人版本；"供给受限 ≠ 泡沫"与 [Pat Gelsinger 的"能源是防泡沫上限"](pat-gelsinger.md)是同一论证的金融版与物理版。

### Sema Amble（a16z 企业软件合伙人）
- **立场与关注点**：headless 软件（数据+逻辑是价值，UI 可选）；创业机会在"两个巨头之间的缝"与"组织内两个职能之间的翻译层"；agent 时代仍需持续采集"例外/context"。
