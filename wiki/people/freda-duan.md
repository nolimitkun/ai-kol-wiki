# Freda（段）

Altimeter Capital 合伙人，硅谷科技基金横跨一二级市场。

## 投资案例

- **一级**：OpenAI、Anthropic、字节跳动
- **二级**：英伟达、存储芯片、Snowflake、Robinhood

## 核心标签

- **Token 经济学分析者**：提出 Token per Task 概念（同一任务不同模型 Token 消耗可差百倍）、$ per Token 类比工业时代 $ per kWh
- **模型公司密切跟踪者**：提供 Anthropic/OpenAI 收入、capex、Gigawatt 经济效益的量化分析
- **组织变革思考者**："把电机塞进蒸汽机"比喻、接力赛→篮球赛的组织模型
- **感性一面的投资圈内人**：坦陈深夜装不上 OpenClaw 痛哭、"95% 信息性对话意义被 AI 掏空"

## 关键判断

- "负向滚雪球"框架修正：Dario 原框架低估了需求弹性——收入增长斜率可以比训练成本增长更陡，公司会突然盈利
- 模型层不是 SaaS，是 usage-based——天然鼓励多 vendor 并用、随时切换
- Code in 的转折点可能已过："马车换汽车"的时刻
- 云生意变差——价值被模型公司分走，自研芯片（TPU/Trainium）成商业化产品
- "投资是效率极低的行业"——agent 理论上 100% 会比人做得好
