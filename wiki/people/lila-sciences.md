# Lila Sciences（Rafa Gómez-Bombarelli & Andy Beam）

**Lila Sciences** 是 Flagship（生物技术孵化器，孵化过 Generate、Moderna 等）体系中"很快分化成异类"的一家——**"neo-lab"**，用 AI 做跨生命科学/化学/材料的科学推理，而非单资产 biotech。合页收录两位联合创始人：**Rafa Gómez-Bombarelli**（物理科学 CSO，前 MIT 材料系教授，早期分子生成 AI/latent space 先驱）与 **Andy Beam**（CTO，前 Harvard 医学 AI、Generate Biomedicines 首任 ML 负责人）。团队含 open-endedness 之父 **Ken Stanley**。本库 AI-for-science 的旗舰美方素材。

## 核心观点

- **科学是"下一个互联网级数据源"**：互联网数据已被压榨殆尽（Ilya："我们只有一个互联网，它是化石燃料"）；**以自然/实验做 verifier 的 RLVR 是该框架的终极版本**，物理实验室即"AI Science Factory = 可规模化 verifier"（[Lila 访谈](../videos/20260716-latent-space-lila-sciences.md) 00:05–00:07）。
- **实验室即 data center、人在 API line 之下**：仪器是图的节点、planar motor 做"实验室 PCI 总线"；一切是 API call（有时机械臂、有时人手）；不做"自动化最大化"，做"token 生成/灵活性最大化"（00:10–00:13）。
- **广度带来深度**：10T 实验验证的科学推理 token（互联网上 order-of-zero）；通用模型常胜过领域专用模型；不做预训练、从 open-weight（NVIDIA/Nemotron）起步（00:32–00:33、01:18–01:21）。
- **模型即价值 / 零 FTE 虚拟创业**：模型是 moat、实验室是 token 生成器；商业形态是客户带明确问题、在平台上跑完整 program（可并行成千上万个"虚拟创业"）；"科学版 Claude Code"帮科学家上抽象阶梯（00:27–00:29、00:49–00:52）。
- **实证成果**：非铂族绿氢电催化剂（"move 37"式惊喜）、量子点、MOF/COF 吸 CO2、in vivo CAR-T（UTR 表达 ~10x、NHP 数据胜 Capstan，2–3 人 6 个月≈5 年 biotech 工作/10% 成本）（00:17–00:49）。
- **瓶颈与中美**：材料"苦甜 scaling"（只有能规模化的才 matter）、材料缺 sim-to-real / 没有 AlphaFold、RL 的 MFU 仅 5–6%；**"美国生物技术输给中国不是创新问题，是监管框架"**（00:42–00:54、01:35–01:39）。
