# Sebastian Raschka & Nathan Lambert

- **背景**: 两位以教学/写作见长的 ML 研究者。**Sebastian Raschka**——《Build a Large Language Model from Scratch》《Build a Reasoning Model from Scratch》作者，主张"从零手搓是最好的学习法"。**Nathan Lambert**——Allen Institute for AI（**AI2**）后训练负责人、**OLMo** 团队，RLHF 权威（著有 RLHF 专书），"RLVR"一词的共同提出者（Tulu 3）。
- **channel slug**: 作为 `lex-fridman` 频道嘉宾收录。

## 立场与关注点

- **架构自 GPT-2 几乎没变**，真正进展在训练配方（pre/mid/post-training）与系统（FP8/FP4）；MoE/MLA/GQA 都是"旋钮"而非范式变化。
- **三条 scaling law（pre-training / RL / inference-time）都还在走**但低垂果实已摘；关键定律是 **RLVR 可 scale、RLHF 不能**。
- **完全开放（data+code）阵营几乎只在西方**（OLMo 为代表）；中国开源生态以超大 MoE 见长、动机是全球 uptake。
- 对 AGI 时间线保守：能力是 **jagged 的**，"一个模型统治一切"的梦正走向专用化；被低估的真实影响是"让全人类知识人人可及"。
- 关注教育与"struggle 的价值"、996/burnout、SF 泡沫等 AI 从业生态问题。

## 呼应与分歧

- 与 [姚顺宇](yao-shunyu.md)：在"纸面趋同、无 winner-take-all、蒸馏/追赶便宜、pre-training 未死"上高度一致，构成本库**美方综述 × 中方一线**的双向印证。
- 与 [Mark Chen](mark-chen.md)、[Noam Brown](noam-brown.md)：共享 jagged frontier、evals 危机、渐进 takeoff 判断。
- 与 [Eric Jang](eric-jang.md)：都从 RL 原理讨论 RLVR/价值函数；Nathan 从 RLHF 侧、Eric 从 self-play 侧互补。
- 与 [Jensen Huang](jensen-huang.md)：都把中国开源视为生态加速器，但两位补充了"美国大厂因安全不付费中国 API → 开源是切入市场的手段"这一机制。
