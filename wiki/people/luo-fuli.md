## 立场与核心观点

- **OpenClaw 是划时代 agent 框架**：一套足够复杂的 agent 框架能弥补大量模型能力短板（3B 端侧模型接入也能干"小模型不该能干"的事）；**智能体框架 = 人与模型之间的"中间层"**，可做得很厚重，前端 UI 是最薄且不关键的一层。
- **开源 + 群体智能 + 自学习**：开源让"全人类群体智慧"共同改进框架、迭代极快；顶尖模型与顶尖 agent 框架必须协同进化。
- **后训练从 Chat 前移到 Agent**：agent 很吃后训练；skills 是预训练的补充（另类信息/组织规范抓不到，只能人机多轮沉淀）；彻底放弃 browsecomp/swebench 等旧 benchmark，靠体感。
- **卡的分配 研究:预训练:后训练 = 3:1:1**：预训练与后训练算力应相当（Chat 时代 3:1~5:1）；卡成了瓶颈（idea→代码很快，卡在并行实验的算力上）。
- **组织平权**：百人团队无组、无层级（"直击"）；层级/规范压制创造力；不分预训练组/后训练组；热爱驱动管理；**环境比经验更重要**（能力 1–4 个月可吸得，招人看好奇心/热爱/多样性/初始上限，越来越招大二大三本科生）。
- **MemoVR 架构**：**hybrid attention（sliding window + full，Pro 达 7:1）取代 MLA** 以适配 agent（省 KV Cache、留计算富裕给 MTP）；**MTP 无幻觉**、只有她们在推理时用；1T 参数是 agent 时代入场券；全模态离散化（音频 RVQ→token，图像进行中）；架构别设太多目标。
- **定价从成本转向价值**：Pro 按"模型产生的价值"（后训练/懂框架的溢价）定价，而非推理成本。
- **开源加速 AGI**：AGI 需分散算力→倒推开源利于框架/芯片/能源；开源不冲突安全（安全归模型、隐私归本地/端侧）。
- **中美**：国内 1T 基座已有数家，反应够快则距 Cloud Ops 4.6 仅 2–3 个月代差且能追上"当代"Claude；国内预训练结构有优势，合力（研究+模型+框架+芯片+能源）非常可能领先。
- **AGI 两年内**：Flash=第一步、现 20%、今年 60–70%；AI 训 AI 是自迭代巅峰/标志节点；工作先被颠覆、生活滞后（需机器人）。
