# 雨森（戴雨森）创投观察第2集

> 张小珺商业访谈录第142期。戴雨森，真格基金管理合伙人。139分钟。whisper medium 转录。
> 第1集（124期）曾预言"2026=Year of R（Return/Research/Reality）"，本次复盘打脸并深入讨论 AI 投资、Harness、组织变革、创业方向。

## 核心观点

### 1. 复盘：什么对了，什么错了

**看对的**（[00:09:37]）：
- OpenAI C端订阅增长停滞：提价难、付费用户已筛选完
- 广告/电商进度远低于预期——"在创新产品里插广告没那么简单"
- 关注return问题的必要性——只是形式变了

**看错的**（[00:10:42]）：
- 完全没料到 Anthropic Coding 会在这么短时间发生突变。"智力突破阈值才有价值——烧到99度也不会有蒸汽机。93度还是99度，提前预计很难。"

**哲学**（[00:05:08]）：strong opinions weakly held。"你的观点有123个原因，当原因变化了，真正的聪明人应该调整看法。市场不会对你说好话。"

### 2. Return 问题没有真正解决

**三段链条**（[00:24:16]）：投入（token）→ 产出（软件）→ 结果（利润）。Anthropic ARR 只是客户的**投入**，不是最终回报。

**降本 vs 增效**（[00:25:17]）：降本（裁员）在发生但只能做一次。增效（新收入）是逐渐的过程——"突然多10倍程序员，收入不一定大涨，因为你不知道做什么新东西。"

**硅谷互相卷**（[00:26:20]）："原来lava宝做网站，现在能做PPT；gamma做PPT，现在能做网站。复制已有功能越来越快，但创新问题不是编程问题。"

**水槽比喻**（[00:29:32]）：上面无数吃螃蟹的人涌进来，下面有人觉得螃蟹不好吃走了。只要进来的人>离开的人，ARR就继续涨。但最终要堵住漏水——Token烧了要能赚回来。

### 3. Coding 是水平能力而非垂直领域

"六个月前连很多senior研究员也认为Coding是垂直方向中的一个。现在发现是horizontal——可以加强办公、医疗、加速研究。"（[00:16:57]）

Anthropic 的组织优势（[00:13:47]）：探索期适合 OpenAI 式自下而上发散；高速赛跑期适合 Anthropic 式自上而下对齐。"Dario 每两周发内部 memo 同步思考——管杀也管埋。OpenAI 经常管杀不管埋。"

### 4. Harness 工程学：壳从哪里来

**模型=CPU，Harness=OS**（[00:53:15]）：三年前做AI应用要自己处理agentic loop/memory/sandbox。现在Cloud Code/Codex帮你处理底层，你像在Windows上写应用一样做skill。

**创新来自第三方**（[00:49:12]）：Manus 率先做 Wide Research（Agent Swarm）、Cloud Code 做 CLI、OpenClaw 做心跳机制+本地权限+IM插入。"这些创新显然不是模型公司做出来的。"OpenClaw的Memory MD机制被研究院一开始不以为然，但用户感受极强。

**壳积累数据反哺模型**（[00:42:07]）：Cursor基于用户数据做post-training出Composer模型——"哪怕只有壳，壳带来的数据也能模型即产品。"

**用户忠诚于Harness**（[00:44:09]）："很多人不断抢救自己的小龙虾，因为存在里面的记忆。模型可以换（Kimi K125替代Opus：90分表现20分价格），但Harness不能换。"

### 5. Agent 网络效应：一个未兑现的可能性

"六个月前你的agent和我的agent没有区别——都跑同样的模型。但现在因为各自的context/skills/个性化沉淀不同，同一个任务给不同人做结果不同。这就有理由互相雇佣。"（[01:00:20]）

"赵小军的agent积累了你所有的采访、准备、思考。我的agent雇佣赵小军的agent，付1000块token费+9000块知识溢价——这是一个agent劳动力市场。"（[01:02:22]）

### 6. AI Native 三阶段

（[01:28:48]）
1. **让人类拥有更多更好的agent**：Cloud Code、OpenClaw、Manus都在做，安装越来越简单
2. **让agent适应人类数字世界**：Agent Mail、Agent Phone、给agent发信用卡（Stripe/Coinbase）、Cloudflare从block AI变服务AI
3. **建设agent专属的数字世界**：agent用CLI/API不用GUI；agent间支付不需要"卡"这个概念

### 7. 下一个字节跳动的公司不会长得像字节跳动

"把信息流+投放+留存搬进AI时代，就是在和上一代冠军竞争。2C做娱乐/杀时间——你上来就要比抖音更好看。但快手字节起来时只需要比用户'发呆刷手机'有意思就行。"（[01:40:00]）

"生产力的提效是10倍100倍。一个AI帮你做Excel vs 原来手打，提效立刻感受得到。而AI短剧 vs 红果短剧，用户不关心是不是AI做的。"（[01:41:02]）

跟风陷阱（[01:57:17]）：可穿戴录音等很多AI硬件是"VC想象出来的需求"。plot很厉害因为它定义了产品形态，后来一堆跟风做各种plot变体就是red flag。

### 8. 组织变革：蒸汽机→电动机的再次上演

与 Freda 的分析不谋而合（[01:37:58]）：蒸汽机工厂围绕一根主轴建设、狭长型；电动机出现后不用主轴了，可以更宽更扁→诞生流水线→生产力飞跃。"技术不是直接带来效率，需要组织配合。组织变化都是人的变化，不是一分钟就变的。"

AI下组织形态（[01:34:56]）：从瀑布式分工（架构/前端/后端/测试/UI）→全栈小团队。投的公司 Slot 从day1让AI参与所有经营环节——build slot using slot。

### 9. 投资判断：投人不是投赛道

四类创始人（[01:16:32]）：小天才、老司机、科学家、操盘手。核心四力：学习力、创造力、领导力、意志力。

"我们不是先看好赛道再找人，而是跟踪优秀的人几年，等他们创业我们就投。"（[01:11:28]）刘松明（RDT系列作者）和丁玲（Simple VLA系列作者）都是跟踪多年、在还没变热时就投的。

"字节系创始人素质肯定很高。但要把字节学会的东西自己颠覆——如果还是信息流刷来刷去加AI小组件，很难有大机会。"（[01:42:03]）

### 10. 硅谷三热 vs 中国方向

**硅谷**（[01:50:11]）：
- Coding大跃进——Meta Leaderboard比谁烧token多
- 世界模型（机器人）——"大家特别想把语言模型范式平移到机器人"
- Auto Research——"AI自我迭代"

**中国**（[01:48:08]）：偏硬——机器人（从硬件→世界模型）、AI应用（生产力+想找下一个字节的）、硬科技（量子/核聚变）

### 11. 个人感悟

- "频繁被打脸是幸福的。不被频繁打脸不是因为你正确，而是因为你没有进步"（[02:15:39]）
- 思想健身房："把思考外包给AI就像一直坐轮椅——腿部肌肉会萎缩。需要思想健身房，刻意练习思考和创新能力"（[02:03:26]）
- Taste也不保险："AI见过那么多好数据，做的PPT比大多数人好看。Taste不是人类的专有阵地。目前剩下的是Agency——问出好问题"（[02:12:37]）
- OOD创新："AI能很好处理within distribution。但有人能创造out of distribution的东西——伽罗华20多岁搞群论。这个目前AI还做不到"（[02:13:37]）
